# ReallySimpleMod

## About the project
A module that trains you how to create modules for [ReallySimpleCMS](https://github.com/ReallySimpleSystems/ReallySimpleCMS).

## Installation
Download the `.zip` archive from GitHub and click 'Install Module' from the modules admin page. A modal window will appear asking you to select an available module. Once selected, the module will be installed by the system.
